CDS Framework

The CDS Framework uses Bootstrap 4 as its base. Anything that Bootstrap 4 offers you can still use with this, it has only been modified to reflect the CDS brand. It uses SASS for the build process which allows us to use things like variables, loops, and mixins.

Getting Started

  1. Install Node.js if not already installed
  2. Open a command window to the root of the project
  3. npm install to get all project dependencies
  4. npm start to start development
  5. The SCSS files are now being watch for changes and will compile each time a change is made

File Stucture

  • CSS
  • Images
  • JS
    Required JavaScript files related to the framework such as Bootstrap and jQuery
    global.js
    Used for initializing plugins or wrapping them with new functionality
  • SCSS
    Contains all framework styling for Bootstrap components
    main.scss
    Entry point for compiling the SCSS used in the framework. When compiled the output is placed in the CSS folder
    variables.scss
    Imports all variables used in the framework including all of Bootstraps variables. A majority of the framework styling/branding can be done modifying the imported files.
    • Mixins
      Reuseable SCSS snippets
    • Plugins
      Third party JavaScript plugin styles
    • Utilities
      Helper classes used to reduce the amount of custom styles
    • Webfonts
      SCSS files related to the webfonts being used
  • Webfonts
    Font files referenced in SCSS/Webfonts

Page Templates

  • Template
    Starter template for new page templates
  • Basic Application
    Simple application template with top navigation
  • Dashboard
    Advanced application template with side navigation and top navigation and slideout panels
